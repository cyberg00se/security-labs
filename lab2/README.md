# Lab2: Weak salt

 В двух словах - нельзя юзать Salsa20 с теми же ключом и nonce.

 Как это работает: шифротекст1 XOR шифротекст2 = сообщение1 XOR сообщение2, потому что одинаковый ключ.
 Если потом делать (сообщение1 XOR сообщение2) XOR с угаданным куском текста из одного из сообщений, то будет получено открытый текст второго из сообщений.

 Таким образом можно делать XOR с популярными штуками (чаще всего это "the", "and", "ing", можно и нужно добавлять пробелы). Здесь было начато с "The ". Потом угадывался кусок побольше (на 1 символ), с которым делалось XOR опять. После XOR с "With this regard" было получено начало фразы "For who would bear the whips and scorns of time,", судя по гуглу это Шекспир.
